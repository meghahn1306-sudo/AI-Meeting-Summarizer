# 🧠 AI Meeting Summarizer

## 📌 Project Overview
An AI-powered web application that converts long meeting conversations or audio recordings into concise, structured summaries using Flask, Ollama (Llama 3), and Whisper.

---

## 🚀 Features
- Paste meeting text and generate instant summary  
- AI-powered summarization using Llama 3 (Ollama)  
- Flask REST API backend  
- Clean and simple frontend UI  
- Audio upload support (coming soon)  
- Speech-to-text using Whisper (planned)  

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Python, Flask  
- AI Model: Ollama (Llama 3)  
- Speech-to-Text: OpenAI Whisper  
- Audio Processing: FFmpeg  
- API Type: REST API  

---

## ⚙️ How It Works
- User enters meeting text in input box  
- Frontend sends request to Flask backend  
- Backend processes text using Llama 3  
- AI generates structured summary  
- Output is displayed on UI  

---

## 🔮 Future Improvements
- Audio recording support  
- Real-time speech-to-text  
- Cloud deployment (AWS / Render / Vercel)  
- Better formatted summaries  
- User authentication system
